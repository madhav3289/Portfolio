# Portfolio - Madhav Gupta

A modern, responsive portfolio website built with React 19, Vite, and Tailwind CSS.

## Features

- ✨ Smooth animations with Framer Motion
- 🎨 Beautiful UI with Tailwind CSS
- 📱 Fully responsive design
- ⚡ Fast performance with Vite
- 📄 Resume download functionality
- 🏆 Certificates showcase
- 💼 Projects showcase
- 🎯 Skills section

## Tech Stack

- **React 19** - UI framework
- **Vite** - Build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Animation library
- **React Icons** - Icon library

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/madhav3289/portfolio.git
cd portfolio
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

The site will be available at `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start the development server
- `npm run build` - Build for production
- `npm run preview` - Preview the production build

## Build Output

The portfolio builds to a static site in the `dist/` directory, ready for deployment to any static hosting service.
